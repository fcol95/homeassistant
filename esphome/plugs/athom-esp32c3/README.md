# Athom ESP32-C3 US Plug For ESPHome Config
Create a new yaml file for each device by copying "plug-athom-esp32c3-example.yaml".
The keys and passwords to replace can be auto-generated by HA esphome builder, by created new device, esp32-c3, and name it accordingly.
Then you can copy the example in their and just keep the generated keys.
You can also set the recovery state by device with switch_restore_mode.

## Flashing instructions with ESPtool:
Already pre-flashed with esphome, plug in a socket and connect to the WIFI the device spawn.
Then, you can use the capture portal web page to connect to your wifi and flash config at http://192.168.4.1.
